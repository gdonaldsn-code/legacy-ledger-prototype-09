# Legacy Ledger (prototype)

Helping families discover and manage the financial accounts of loved ones after
they've passed — find unclaimed assets and close accounts with dignity.

## Stack

- Vite + React 18 + TypeScript
- shadcn-ui (Radix primitives) + Tailwind CSS
- React Router, TanStack Query, React Hook Form + Zod

This is currently a frontend-only prototype — no backend/database is wired up yet.

## Getting started

```sh
npm i
npm run dev
```

Other scripts: `npm run build`, `npm run lint`, `npm run preview`.
